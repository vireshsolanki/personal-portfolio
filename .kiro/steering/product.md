# Product Overview

This is a personal portfolio website for Viresh Solanki, an AWS Cloud & DevOps Engineer. The portfolio showcases professional experience, technical skills, projects, and client testimonials.

## Purpose

- Display professional profile and work experience
- Showcase cloud infrastructure and DevOps projects
- Provide contact information for freelance opportunities
- Demonstrate technical expertise in AWS, Kubernetes, Docker, and CI/CD

## Key Features

- Dark theme with neon green (#00FF87) accent color
- Smooth scroll animations using IntersectionObserver
- Fully responsive design (mobile, tablet, desktop)
- SEO optimized with meta tags
- Performance optimized (Lighthouse 95+)
- Single-page application with section navigation

## Content Management

All portfolio content is centralized in `src/data/portfolio.js` for easy updates. This includes personal info, projects, skills, experience, and testimonials.
